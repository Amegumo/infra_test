# 想定環境

# Version
    Terraform v1.2.9

# terraform コマンド
1. プラグインのダウンロード

    `terraform init`

1. terraformのフォーマット

    `terraform fmt`

1. terraformの検証

    `terraform validate`

1. 作成した環境のセットアップ
    
    `terraform apply` -> yes

1. 作成した環境の削除

    `terraform destroy` -> yes
